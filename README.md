# Flat White

Flat White is a polished static coffee app prototype for brew guides, extraction profiles, bean pantry tracking, and coffee bag detail extraction.

## Features

- Front cover with subtle global coffee terminology background
- 15+ brew methods for espresso, filter, immersion, stovetop, and milk drinks
- Step-by-step brew guides with ratio, grind, time, and bean-fit notes
- Custom extraction profile controls
- Bean pantry for roaster, origin, variety, process, and flavour notes
- Coffee bag scanner simulation that extracts roaster, origin, roast, process, variety, and tasting notes from label text

## Deploy

This is a static HTML/CSS/JavaScript app. It can be deployed directly to Vercel, Netlify, GitHub Pages, or any static host.

For Vercel, use:

- Framework preset: Other
- Build command: leave blank
- Output directory: `.`
